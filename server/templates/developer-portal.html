Quick Start

Get started with the CryptoEats API in minutes. Create an account, get your API keys, and start building.

1. Create API Keys

curl -X POST BASE_URL_PLACEHOLDER/api/developer/keys \
  -H "Authorization: Bearer YOUR_JWT_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"name": "My App", "tier": "free"}'

Save your secret key immediately — it's only shown once. You'll receive a public key (ce_pk_...) and secret key (ce_sk_...).

2. Make Your First Request

curl BASE_URL_PLACEHOLDER/api/v1/restaurants \
  -H "X-API-Key: ce_pk_your_public_key"

3. Set Up Webhooks

curl -X POST BASE_URL_PLACEHOLDER/api/developer/webhooks \
  -H "Authorization: Bearer YOUR_JWT_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "url": "https://your-app.com/webhooks/cryptoeats",
    "events": ["order.created", "delivery.completed"]
  }'

Authentication

CryptoEats uses two authentication methods depending on the context.

API Key Authentication (Platform API)

For accessing the Platform API v1 endpoints, include your API keys in the request headers:

X-API-Key: ce_pk_your_public_key
X-API-Secret: ce_sk_your_secret_key  (required for write operations)

JWT Bearer Token (Developer Management)

For managing API keys, webhooks, and configurations, use JWT authentication:

Authorization: Bearer eyJhbGciOiJIUzI1NiIs...

Rate Limiting

Rate limits are applied per API key based on your tier. Response headers include:

X-RateLimit-Limit: 1000        # Daily request limit
X-RateLimit-Remaining: 950     # Remaining requests
X-RateLimit-Reset: 2026-02-08T00:00:00Z  # Reset time

Signature Verification

// Node.js verification example
const crypto = require('crypto');

function verifySignature(payload, signature, secret) {
  const expected = crypto
    .createHmac('sha256', secret)
    .update(payload)
    .digest('hex');
  return `sha256=${expected}` === signature;
}

// In your webhook handler:
app.post('/webhooks/cryptoeats', (req, res) => {
  const signature = req.headers['x-cryptoeats-signature'];
  const isValid = verifySignature(
    JSON.stringify(req.body),
    signature,
    'whsec_your_webhook_secret'
  );
  if (!isValid) return res.status(401).send('Invalid signature');
  // Process event...
  res.status(200).send('OK');
});

SDKs

Use our client libraries to integrate CryptoEats into your application.

// npm install @cryptoeats/sdk

const CryptoEats = require('@cryptoeats/sdk');

const client = new CryptoEats({
  apiKey: 'ce_pk_your_public_key',
  apiSecret: 'ce_sk_your_secret_key',
  sandbox: true,
});

// List restaurants
const restaurants = await client.restaurants.list({ featured: true });
console.log(restaurants.data);

// Get restaurant menu
const menu = await client.restaurants.menu('restaurant-id');

// Submit inbound order (from your POS/store)
const order = await client.integrations.createInboundOrder({
  externalOrderId: 'SHOP-12345',
  source: 'shopify',
  customerName: 'John Doe',
  deliveryAddress: '123 Main St, Miami, FL 33101',
  items: [
    { name: 'Margherita Pizza', price: 14.99, quantity: 2 },
    { name: 'Craft Beer', price: 8.99, quantity: 1, isAlcohol: true },
  ],
  subtotal: 38.97,
  total: 41.99,
});

// Calculate tax
const tax = await client.tax.calculate({ subtotal: 38.97, rate: 0.07 });

// Mint NFT reward
const nft = await client.nft.mint({ wallet: '0x...', type: 'milestone' });

// Set up webhook listener
client.webhooks.on('order.delivered', (event) => {
  console.log('Order delivered:', event.data.orderId);
});
